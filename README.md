# Masterthesis_Hans_Amberg_2598617
Dies ist der verwendete Code zur Berechnung der Ergebnisse meiner Masterthesis "Möglichkeiten und Grenzen des Machine Learnings in der partizipativen und nachhaltigen Innenstadtplanung"

Zur Berechnung wird der Datensatz der Studie zur Transformation der Innenstädte von Pfnür & Rau (2024) benötigt, welcher beim Lehrstuhl für Immobilienwirtschaft & Baubetriebwirtschaftslehre der TU Darmstadt angefragt werden kann.

Die Berechnungen wurden in vier Codes aufgeteilt:
1. Regression: Random Forest + OLS
2. Regression: XGBoost
3. Klassifikation: Random Forest
4. Klassifikation: XGBoost + multinomial + Shapley
